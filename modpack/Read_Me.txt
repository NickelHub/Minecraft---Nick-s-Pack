Minecraft - Variety Revelations Modpack
made by NickelCraft

Installation:
	x Drag and drop the "mods", "scripts" and "config" folders in your game directory.
	x Install Minecraft Forge. It is found in the forge folder.
	x The "mods_JEI_extra" folder and the "mods_atmosphere" folder include JEI addons and asthetic mods. You need to place them in the mods folder if you want to use them. Note that this can decrease performance.
	x Start the game with 6GB+ of RAM
