Minecraft - Nick's Modpack
made by NickelCraft

Installation:
	x Drag and drop the "mods", "scripts" and "config" folders in your game directory.
	x Install Minecraft Forge. Version is found in forge folder.
	x Start game with 8GB+ of RAM
	
Installation Server:
	x Drag and drop the "mods", "scripts" and "config" folders in your game directory.
	x Install Minecraft Forge. Version is found in forge folder.
		x Delete following mods from sever
			x AppleSkin
			x Better Foliage
			x Better Placement
			x Controlling
			x Item Got
			x Item Zoom
			x JEI Integration
			x JourneyMap | game crashes if you change the dimension due to a bug
			x ModNameTooltip
			x Optifine
			x SmoothFont
	x Start server with at least 8GB+ of RAM
